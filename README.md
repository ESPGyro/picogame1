* 搜索 https://github.com/ESPGyro/picogame

## 編輯此專案

編輯 MakeCode 中的儲存庫。

* 開啟 [https://makecode.microbit.org/](https://makecode.microbit.org/)
* 按一下**匯入**，然後按一下**匯入 URL**
* 貼上 **https://github.com/ESPGyro/picogame** 並按一下匯入
